Sale Order Multi Product Selection

Key Features:

  • ★   Add multiple lines to a sale order line from sale order at a time on single click .
  • ★   Create sale quotation from product variant view with multiple products.

Overview

This module allows you to add multiple products in sale order line on a single click. You can also create sale quotation from product variant Tree view with selected multiple products very quickly.

From Sale Quotation View

Added one button in sale quotation view i.e "Select Product"

The wizard will open after click on "Select Product" button.

You will get the selection of product when you click on add an item

Selected Product will be added in wizard.

All the selected line will be added in order lines if you will click on apply.

From Product Variant View.

Select product as much as you want from the product variant tree view and click on action menu. There you will see a create quotation menu.

Wizard for create quotation will open.

Quotation will be created with selected products.
